# QuickThumb Bot
Ce bot Telegram permet de changer la miniature de vos fichiers vidéo en masse.
Voici les commandes principales :

- `/set_thumb` : Envoyer une photo à utiliser comme miniature
- `/del_thumb` : Supprimer toutes les miniatures enregistrées
- `/seq_start` : Commencer l'envoi de fichiers à traiter
- `/seq_stop` : Terminer l'envoi et commencer le traitement
- `/pause` : Mettre en pause le traitement
- `/resume` : Reprendre le traitement
